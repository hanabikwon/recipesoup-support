⏺ Recipesoup - 1인 개발 MVP 기획서

  소중한 사람을 위한 요리 기록 서비스

  ---
  🎯 기획 의도

  왜 이 앱을 만드는가?

  개인적 동기

  "요리를 사랑하는 남편을 보며 시작한 프로젝트"

  요리하는 사람의 마음을 담고 싶었다
  - 사랑하는 사람을 위해 요리할 때 콧노래를 부르는 모습
  - 맛있게 먹는 모습을 보며 행복해하는 순간
  - 레시피를 탐구하고 연구하는 진지함
  - 그런 마음들을 영원히 기록할 수 있는 공간

  실제 불편함 (Pain Points)

  문제 1: 레시피 파편화
    "맛있게 해먹었던 그 레시피 어디 있지?"
    - 네이버 블로그 → 비공개 전환
    - 유튜브 → 영상 삭제됨
    - 인스타 → 계정 사라짐
    - 카톡 메모 → 묻혀서 못 찾음

  문제 2: 나만의 황금 비율
    "원래 레시피에서 설탕은 반으로, 간장은 1스푼 더"
    - 매번 미세 조정하며 완성한 우리집 비율
    - 원본 + 나만의 조정사항 함께 기록 필요
    - 시행착오의 역사도 소중한 기록

  문제 3: 시간 부족
    "바쁜데 이 긴 블로그 글 언제 다 읽어"
    - 광고와 서론 사이 숨은 핵심 레시피
    - 10분 영상에서 정작 필요한 건 30초
    - AI로 핵심만 빠르게 요약 필요

  해결 방향

  "모든 레시피를 한 곳에, 빠르고 영구적으로"
  - 🔗 URL 저장 + AI 요약으로 시간 절약
  - 📝 나만의 조정사항 기록
  - 💝 요리에 담긴 이야기 아카이빙
  - 🔒 영구 보관 (로컬 저장)

  ---
  👥 타겟 사용자

  Primary Target: "요리하는 마음을 기록하는 사람들"

  핵심 특징

  공통점:
    - 요리를 사랑의 표현으로 여김
    - 레시피를 연구하고 발전시킴
    - 바쁜 일상 속 효율적 기록 원함
    - 소중한 순간을 남기고 싶어함

  연령대: 25-40세
  성별: 무관 (요리하는 모든 사람)
  라이프스타일:
    - 평일 바쁨, 주말 요리
    - 디지털 네이티브
    - 기록과 아카이빙 선호

  세부 페르소나

  1. "아내를 위해 요리하는 준호" (34세, 남성) 🧑‍🍳

  배경:
    - IT 개발자, 결혼 3년차
    - 주말마다 아내를 위한 특별 요리
    - 유튜브로 요리 공부하는 열정파
    - 레시피 연구를 취미로 즐김

  Pain Points:
    "유튜브 10분 영상 다 볼 시간이 없어"
    "내가 조정한 레시피 버전 관리가 필요해"
    "아내가 좋아했던 그 요리 뭐였더라"

  사용 시나리오:
    금요일 저녁 - 주말 요리 계획
    1. 유튜브 URL 복사 → 앱에 붙여넣기
    2. AI가 핵심 레시피만 요약 (30초)
    3. "고추장 1스푼 추가" 메모
    4. 감정: "결혼기념일 특별 요리"
    5. 아내 반응: ⭐⭐⭐⭐⭐

  핵심 니즈:
    - 빠른 레시피 요약 (시간 절약)
    - 버전별 기록 (실험 정신)
    - 반응 기록 (피드백 관리)

  2. "가족을 위해 요리하는 수진" (35세, 여성) 👩‍🍳

  배경:
    - 육아맘 5년차
    - 엄마 레시피 복원 프로젝트
    - 아이 영양 밸런스 고민
    - 시부모님 입맛도 고려

  Pain Points:
    "엄마 레시피는 '대충 한 숟갈'이라니"
    "아이가 잘 먹었던 레시피 기록하고 싶어"
    "명절 요리 작년에 어떻게 했더라"

  사용 시나리오:
    평일 오후 - 저녁 메뉴 준비
    1. "유아식" 태그로 검색
    2. 지난달 잘 먹었던 레시피 확인
    3. 재료 체크하며 장보기 리스트
    4. 완성 사진 + "오늘도 완밥!"
    5. 시어머니 레시피 추가 저장

  핵심 니즈:
    - 가족 구성원별 선호도 기록
    - 영양 정보 메모
    - 계절/명절 요리 아카이빙

  3. "나를 위해 정성껏 요리하는 민지" (30세, 여성) 👩‍💻

  배경:
    - 스타트업 마케터
    - 1인 가구 3년차
    - 자기 돌봄으로서의 요리
    - 인스타 대신 진짜 기록

  Pain Points:
    "혼자 먹는데도 제대로 하고 싶어"
    "블로그 레시피 너무 양이 많아"
    "나만의 1인분 레시피 필요해"

  사용 시나리오:
    퇴근 후 저녁 - 나를 위한 요리
    1. 냉장고 재료 확인
    2. 저장된 1인분 레시피 검색
    3. "피곤한 날" 태그 클릭
    4. 15분 완성 레시피 선택
    5. "오늘도 잘 챙겨먹었다" 기록

  핵심 니즈:
    - 1인분 레시피 특화
    - 컨디션별 요리 분류
    - 자기 돌봄 기록

  4. "부모님을 위해 요리하는 현우" (28세, 남성) 👨‍🍳

  배경:
    - 대학원생, 본가 거주
    - 고령 부모님 건강식 담당
    - 당뇨/혈압 고려한 레시피
    - 주말마다 일주일치 반찬

  Pain Points:
    "저염식 레시피 찾기 어려워"
    "부모님 입맛에 맞춘 조정 필요"
    "건강 정보도 함께 기록하고 싶어"

  사용 시나리오:
    일요일 오전 - 일주일 반찬 준비
    1. "저염식" 폴더 열기
    2. 지난주 잘 드셨던 메뉴 체크
    3. 새 레시피 URL → AI 요약
    4. "소금 대신 레몬즙" 메모
    5. 부모님 건강 상태 함께 기록

  핵심 니즈:
    - 건강 레시피 관리
    - 가족 피드백 기록
    - 영양 정보 추적

  💡 핵심 컨셉

  서비스 정의

  "바쁜 당신을 위한 스마트한 레시피 아카이빙"

  Core Value Proposition

  1. ⚡ AI 요약 (Time Saver)

  왜 중요한가:
    "바쁜 현대인에게 시간은 금"
    - 10분 영상 → 30초 요약
    - 긴 블로그 → 핵심만 추출
    - 광고 제거 → 레시피만

  기능:
    - URL 입력 즉시 AI 분석
    - 재료와 조리법 자동 정리
    - 핵심 팁만 하이라이트

  2. 📝 나만의 진화 (Personalization)

  왜 중요한가:
    "완벽한 레시피는 진화한다"
    - 원본 + 내 조정사항
    - 시행착오 히스토리
    - 우리집 황금 비율

  기능:
    - 버전 관리
    - 메모 무제한
    - Before/After 비교

  3. 💝 감정 기록 (Emotional Context)

  왜 중요한가:
    "요리는 사랑의 표현"
    - 누구를 위해 만들었는지
    - 그때의 기분과 상황
    - 받는 사람의 반응

  기능:
    - 감정 태그
    - 스토리 메모
    - 평점과 피드백

  4. 🔒 영구 보관 (Forever Safe)

  왜 중요한가:
    "소중한 레시피는 영원히"
    - 링크 깨짐 걱정 없음
    - 로컬 저장 안정성
    - 내 핸드폰이 내 레시피북

  기능:
    - 오프라인 접근
    - 자동 백업
    - 스크린샷 보관

  🚀 MVP 범위 (8주 개발)

  바쁜 사람을 위한 핵심 기능

  ✅ Phase 1: MVP (필수)

  1. 스마트 레시피 저장 (핵심)
     - URL → AI 요약 (30초)
     - 스크린샷 자동 저장
     - 나만의 메모 추가
     - 사진 갤러리

  2. AI 파워 기능 (차별화)
     - URL 스크래핑 + 요약
     - 사진 → 레시피 생성
     - 재료 자동 추출
     - 1인분 변환

  3. 빠른 검색
     - 비주얼 카드 뷰
     - 최근/즐겨찾기
     - 태그 필터

  4. 안전한 보관
     - 100% 로컬 저장
     - 링크 깨져도 안전
     - 백업 준비

  🔄 Phase 2: 확장

  - 가족 공유
  - 쇼핑 리스트
  - 영양 정보
  - 타이머/알람
  - 버전 히스토리

  ---
  📱 UI/UX 설계

  디자인 컨셉: "모던 빈티지 레시피북"

  키워드:
    - 따뜻하지만 효율적
    - 빈티지 감성 + 모던 기능
    - 사진 중심 비주얼
    - 직관적 인터랙션

  핵심 사용 플로우

  1. 빠른 저장 (10초)

  URL 복사 → 앱 오픈 → 자동 붙여넣기 감지
  → "AI가 요약 중..." (3초)
  → 요약된 레시피 확인
  → 내 메모 추가 (선택)
  → 저장 완료!

  2. 요리하며 보기

  레시피 선택 → 쿡 모드
  → 화면 꺼짐 방지
  → 내 메모 하이라이트
  → 타이머 연동 (선택)

  3. 빠른 검색

  홈 → 사진으로 바로 인식
  또는
  검색 → "생일" → 관련 레시피
  또는
  태그 → #빠른요리 → 15분 레시피

  화면 구성 (심플)

  홈 (갤러리)
  ├── 검색바
  ├── 레시피 카드 그리드
  │   └── 사진 + 제목 + 시간
  └── FAB
      ├── 📎 URL로 추가 (AI 요약)
      ├── 📷 사진으로 추가
      └── ✏️ 직접 작성

  Special Features:
    - URL 스크래핑
    - 오프라인 우선
    - 자동 백업 준비
